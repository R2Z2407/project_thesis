Pemantauan Kondisi Mesin Milling untuk Peningkatan Efisiensi

Mesin milling, yang penting untuk produksi komponen presisi, rentan mengalami kegagalan tak terduga yang sulit dideteksi secara manual. Kegagalan ini menyebabkan biaya perbaikan, penurunan produksi, dan risiko kualitas.

02

Kelemahan Pemeliharaan Preventif Tradisional

Pemeliharaan preventif, yang mengandalkan jadwal tetap, tidak efektif karena tidak mempertimbangkan kondisi aktual mesin. Hal ini menyebabkan mesin masih berisiko mengalami kerusakan mendadak di luar jadwal perawatan.

Solusi

Deteksi Anomali Melalui Pengklasifikasian Berbasis Model Mesin Pembelajaran.

Model tersebut menganalisis dari beberapa kondisi data sensor dari mesin milling secara real-time untuk mendeteksi anomali. Pembuatan model menggunakan bahasa pemprograman python. Model dibuat melalui pendekatan klasifikasi biner. Hasil analisis model tersebut berupa penyataan anomali dan normal. Selain itu, model dilakukan penguji tingkat efektif mesin pembelajaran diantaranya random forest,extreme gradient boosting, support vector machine, dan artificial neural network. Pengujian model melalui beberapa pengukuran confusion matrix, precition, recall, f1-score, Precision-Recall Curve, G-Mean, dan Matthews Correlation Coefficient.

Model Mesin Pembelajaran

Model random forest, extreme gradient boosting, support vector machine, dan artificial neural network menganalisis air temperature, air process, torque, dan rotational speed untuk memberikan pernyataan mesin mengalami kondisi anomali atau normal. Setiap model memiliki parameter dan cara mempelajarinya tersendiri.

Random Forest

Random Forest adalah sebuah metode algoritma ML yang termasuk kedalam kategori yang menggabungkan beberapa pohon keputusan. Model ini memiliki bootstrap aggregating (bagging) untuk mengurangi varian dan meningkatkan stabilitas prediksi. Berikut Parameter dari model ini:

  1. n Estimator: 100
  2. Criterion: gini
  3. Min_samples_split : 2
  4. Min_samples_leaf : 2
  5. Max_features : sqrt
  6. Max_depth : none
  7. Bootstrap : true
  8. Class weight : balanced
  9. Random state : 42

Extreme Gradient Boosting

Extreme Gradient Boosting (Xgboost) adalah sebuah metode algoritma ML dalam kategori ensemble learning, khususnya boosting untuk meningkatkan kecepatan model. Model ini menerapkan teknik regularisasi sehingga mampu mencegah overfitting, yang sering menjadi tantangan dalam model yang kompleks. Berikut Parameter dari model ini:

  1. Learning rate: 0.22
  2. n Estimator: 600
  3. Gamma : 2
  4. Max_depth : 4
  5. Min_child_weight : 3
  6. Reg_lambda : 1
  7. Reg_alpha : 2
  8. Subsample : 0.7
  9. Colsample_bytree : 0.7
  10. Scale_pos_weight : balanced
  11. Random state : 42

Support Vector Machine

Support Vector Machine (SVM) adalah sebuah metode algoritma pembelajaran mesin yang berfokus pada pencarian hyperplane optimal yang memisahkan kelaskelas dalam ruang fitur. SVM bekerja dengan membangun model yang memaksimalkan margin antara dua kelas. Margin didefinisikan sebagai jarak terdekat antara titik data dari masing-masing kelas ke hyperplane. Berikut Parameter dari model ini:

  1. c: 99.78
  2. Gamma : 0.99
  3. Probability : 4
  4. Class weight : 3
  5. Kernel : 1
  6. Random state : 42

Artificial Neural Network

Artificial Neural Network (ANN) adalah model komputasi yang terinspirasi oleh jaringan saraf biologis, yang terdiri dari elemen-elemen pemrosesan sederhana yang disebut neuron. Neuron ini saling terhubung dan bekerja secara paralel untuk menyelesaikan berbagai tugas. ANN terdiri dari tiga komponen utama: lapisan masukan, lapisan tersembunyi, dan lapisan keluaran. Berikut Parameter dari model ini:

  1. Input layer: 4 neuron
  2. Hidden layer 2 layer
  3. Hidden layer pertama : 64 neuron
  4. Hidden layer kedua : 16 neuron
  5. Dropout dalam hidden layer : 0.2
  6. Output layer : 1 neuron
  7. Optimization: 4 neuron
  8. Epoch 2 layer
  9. Batchsize: 4 neuron
  10. Random state 2 layer
  11. loss: 4 neuron

Pengujian

Pengujian setiap model dilakukan untuk melihat tingkat efektivitasnya. Tingkat efektivitas dilihat melalui precition,recall, f1-score, precision-recall curve(PRC), matthew correlation coefficient(MCC), dan G-mean. Hal tersebut merujuk pada kegunaan model tersebut dalam mendeteksi keberadaaan titik normal dan anomali.

Precision

Precision adalah proporsi hasil prediksi positif yang benar-benar positif. Semakin tinggi nilai precision, semakin sedikit prediksi positif yang salah.

Recall

Recall adalah kemampuan model untuk mendeteksi semua kasus positif yang sebenarnya. Nilai recall tinggi berarti model jarang melewatkan kasus positif.

F1-Score

F1-Score adalah harmonic mean dari precision dan recall, memberikan keseimbangan antara kedua metrik tersebut. Metrik ini berguna ketika kita perlu menyeimbangkan precision dan recall, terutama pada dataset yang tidak seimbang (imbalanced). F1-score lebih baik daripada akurasi biasa karena tidak dipengaruhi oleh mayoritas kelas negatif.

Precision-Recall Curve

Precision-Recall Curve (PRC) adalah grafik yang menunjukkan hubungan antara precision dan recall pada berbagai threshold klasifikasi. Kurva ini lebih informatif daripada ROC curve untuk dataset tidak seimbang karena fokus pada kelas minoritas. Area Under the Curve (AUC) dari kurva ini (AUC-PR) digunakan untuk mengevaluasi performa keseluruhan model.

Matthew Correlation Coefficient

Matthews Correlation Coefficient (MCC) adalah metrik korelasi antara prediksi model dan label aktual, yang menghasilkan nilai antara -1 hingga 1. Nilai 1 menunjukkan prediksi sempurna, 0 menunjukkan prediksi acak, dan -1 menunjukkan prediksi selalu salah. MCC sangat tangguh terhadap ketidakseimbangan kelas dan memberikan gambaran seimbang tentang performa model.

G-mean

G-mean adalah akar kuadrat dari hasil kali recall (sensitivitas) dan spesifisitas... (deskripsi lengkap)

Hasil

Berdasarkan hasil pengujian mengungkapkan bahwa model ANN unggul dengan nilai-nilai yang terbaik berdasarkan evaluasi dibandingkan metode lain. Hal ini terlihat dari evaluasi classification report pada kelas anomali yaitu nilai presisi 0,81, nilai recall 0,91, dan nilai f1-score 0,85. Jika melihat kurva PRC, nilai ANN lebih unggul. Selain itu, nilai g-mean dan MCC unggul yaitu masing-masing bernilai 0,95 dan 0,85.
